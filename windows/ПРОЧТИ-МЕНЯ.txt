yt2premiere — скачивание YouTube в высоком качестве и конвертация для Premiere Pro
=================================================================================
ВЕРСИЯ ДЛЯ WINDOWS (10/11, 64-бит)

КАК ПОЛЬЗОВАТЬСЯ (просто)
------------------------
1. Дважды кликни по файлу:  «Скачать видео с YouTube.bat»
2. Вставь ссылку (правый клик в окне = вставить).
3. Укажи папку для скачивания (или нажми Enter — по умолчанию «Видео\YouTube»).
4. Выбери формат: 1 = MP4, 2 = ProRes (лучший для монтажа), 3 = MP3.
5. Готово — папка с результатом откроется сама.

ПЕРВЫЙ ЗАПУСК
-------------
При самом первом запуске программа сама скачает два инструмента
(yt-dlp.exe и ffmpeg.exe, ~80 МБ) в подпапку bin\ рядом с собой.
Нужен интернет. Дальше они уже не качаются.

ВОЗМОЖНЫЕ ПРЕДУПРЕЖДЕНИЯ
------------------------
* «Windows защитила ваш компьютер» (SmartScreen):
  нажми «Подробнее» -> «Выполнить в любом случае».
* Антивирус иногда ложно ругается на yt-dlp.exe — это известное
  ложное срабатывание; добавь папку в исключения, если потребуется.

КОМАНДНАЯ СТРОКА (для продвинутых)
---------------------------------
В PowerShell, находясь в этой папке:

  .\yt2premiere.ps1 "ССЫЛКА"                         MP4, макс. качество
  .\yt2premiere.ps1 -Format prores "ССЫЛКА"          ProRes .mov для монтажа
  .\yt2premiere.ps1 -Out "D:\Footage" "ССЫЛКА"       своя папка
  .\yt2premiere.ps1 -Max 1080 url1 url2              пачкой, не выше 1080p
  .\yt2premiere.ps1 -Mp3 "ССЫЛКА"                    только звук

ЕСЛИ ЧТО-ТО СЛОМАЛОСЬ
---------------------
YouTube периодически меняет раздачу. Программа лечит это сама
(обновляет yt-dlp и пробует другой клиент). Если всё равно не качает —
запусти ещё раз: при сбое она принудительно обновит yt-dlp.

АППАРАТНОЕ УСКОРЕНИЕ
--------------------
MP4 кодируется на видеокарте, если есть: NVIDIA (nvenc), Intel (qsv)
или AMD (amf). Если ничего не найдено — программный кодек (медленнее,
но качество то же).
