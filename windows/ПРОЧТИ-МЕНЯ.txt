yt2premiere — скачивание YouTube в высоком качестве и конвертация для Premiere Pro
=================================================================================
ВЕРСИЯ ДЛЯ WINDOWS (10/11, 64-бит)

ПРИЛОЖЕНИЕ С ОКНОМ (рекомендуется)
---------------------------------
Дважды кликни:  «Приложение yt2premiere.bat»
Откроется окно:
  1. Вставь ссылку (Ctrl+V) -> выбери качество и формат -> «＋ Добавить в очередь».
     Можно добавить сколько угодно роликов, у каждого своё разрешение.
  2. Укажи папку (или оставь «Видео\YouTube»).
  3. Нажми «⬇ Скачать всю очередь» — ролики качаются по очереди,
     статус каждого виден в списке, внизу — журнал.
  4. По завершении папка откроется сама.

ПЕРВЫЙ ЗАПУСК
-------------
При самом первом скачивании программа сама загрузит yt-dlp.exe и ffmpeg.exe
(~80 МБ) в подпапку bin\. Нужен интернет. Дальше уже не качаются.

ВОЗМОЖНЫЕ ПРЕДУПРЕЖДЕНИЯ
------------------------
* «Windows защитила компьютер» (SmartScreen): «Подробнее» -> «Выполнить в любом случае».
* Антивирус иногда ложно ругается на yt-dlp.exe — известное ложное срабатывание.

КАЧЕСТВО (РАЗРЕШЕНИЕ)
---------------------
Для каждого ролика в очереди можно выбрать: Макс. / 2160p / 1440p / 1080p / 720p / 480p / 360p.
Формат: MP4 (универсально), ProRes (.mov, лучший для монтажа), MP3 (только звук).

ЕСЛИ ОКНО НЕ ОТКРЫЛОСЬ
----------------------
Можно воспользоваться простой версией без окна:
  «Скачать видео с YouTube.bat»  — спросит ссылку и формат в текстовом виде.
Или командой в PowerShell (в этой папке):
  .\yt2premiere.ps1 -Out "D:\Footage" -Max 1080 "ССЫЛКА"

ФАЙЛЫ
-----
  Приложение yt2premiere.bat       — запуск окна (GUI)
  yt2premiere-gui.ps1              — само окно
  yt2premiere.ps1                  — движок (скачивание + конвертация)
  Скачать видео с YouTube.bat      — простая версия без окна (запасная)

АППАРАТНОЕ УСКОРЕНИЕ
--------------------
MP4 кодируется на видеокарте, если есть: NVIDIA (nvenc) / Intel (qsv) / AMD (amf),
иначе программный кодек (медленнее, качество то же).
